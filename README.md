# OpenECS

If you are an AI agent or automated coder, read `AGENT.md` first.

## Repo Layout

- `openecs-js/` - the importable ECS runtime package
- `openecs-demo/` - the browser demo that uses the runtime
- `openecs-docs/` - repo documentation and notes

## Where To Start

- For JavaScript runtime work, read `openecs-js/README.md`.
- For browser or CDN import work, also read `openecs-js/WEBAGENT.md`.
- For package-level coding rules, read `openecs-js/AGENT.md`.
